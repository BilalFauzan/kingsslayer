# kingsslayer

## Run the app locally
### Install streamlit via pip
```
pip install streamlit
```
### Install streamlit via conda
```
conda install -c conda-forge streamlit
```
### Run
```
https://raw.githubusercontent.com/BilalFauzan/kingsslayer/main/minyak.py
```
## Required packages
- Streamlit
- Pandas
- Numpy
- Matplotlib
